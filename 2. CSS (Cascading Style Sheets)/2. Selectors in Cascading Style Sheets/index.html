Selectors in CSS

CSS selectors target the HTML elements on your pages, allowing you to add styles based on their ID, class, type, attribute, and more.

Various types of selectors are: ID selector, Class selector, universal selector, tag selector etc.

An ID selector uniquely identifies an HTML element. The symbol used is #

A class selector is used when we want to apply the same style to multiple HTML elements. The symbol used is .

A universal selector targets all HTML elements in the document. The symbol used is *

A tag selector targets all HTML elements with the same tag name. The symbol used is the name of the tag itself.
